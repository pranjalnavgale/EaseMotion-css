# ⚡ EaseMotion CSS - Scroll Reveal Utility

A utility to reveal layout elements smoothly as they enter the browser viewport frame using performance-optimized rendering pipelines.

## Installation & Setup

1. Include `style.css` alongside your project files.
2. Add the `.ease-scroll-reveal` class wrapper to your elements:

```html
<div class="ease-scroll-reveal">
  <!-- Elements to reveal -->
</div>
